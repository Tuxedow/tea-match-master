# 奶茶匹配大师

根据用户的身份、年龄、职业和爱好，自动匹配适合的奶茶品牌和奶茶单品。

![奶茶匹配大师](https://img.shields.io/badge/奶茶-匹配-blue)

## 功能特点

- 🎰 二次元游戏化风格界面
- ⭐ 等级系统（萌新奶茶师 → 奶茶大师）
- 🏆 成就徽章系统
- 📸 截图分享功能
- 📋 一键复制结果
- 💾 下载报告

## 在线访问

访问地址：https://tuxedow.github.io/tea-match-master/

## 本地运行

直接在浏览器中打开 `index.html` 即可。

## 技术栈

- HTML5
- CSS3 (动画、响应式)
- JavaScript (ES6+)
- html2canvas (截图功能)
